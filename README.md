# Leadership Playbooks

Process playbooks for product, engineering leadership, and organizational decisions.

These playbooks document *how to lead* — not how to implement. For implementation-level
processes (testing strategies, deployment runbooks, ADR writing conventions), see
.

| Topic | Lives here |

| Architecture Review Board governance | Yes |

| Org design, hiring, performance frameworks | Yes |
| Mentorship and career ladders | Yes (professional development overlap) |
| How to write an ADR | No — engineering-playbooks |
| Testing strategy | No — engineering-playbooks |
| Deployment runbooks | No — engineering-playbooks |
